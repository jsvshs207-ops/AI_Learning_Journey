## 项目介绍
> 非计算机学科学生因科研需要学习机器学习及使用Python进行数据分析的学习记录。其中《机器学习》及《深度学习》部分为九天&菜菜课程的学习记录笔记，九天菜菜的课程私以为是机器学习教学领域最强的课，讲解的非常清楚、简单易懂。特别推荐这个课程
### 1、Pands系列学习
这个仓库主要是记录pandas、matplotlib、numpy等数据处理、分析的库。这些库是Python数据分析及机器学习等的必备工具。
### 2、机器学习系列学习
这个仓库主要记录Sklearn库的学习，Sklearn是涵盖众多经典机器算法的强大机器学习库，利用Sklearn可以迅速搭建机器学习模型。
### 3、深度学习系列学习
这个仓库主要记录神经网络方面的学习，神经网络是深度学习算法的基石，里面详细介绍了神经网络从底层算法原理到手搓神经网络，再到利用Pytorch迅速搭建模型的一系列知识。
### 4、Pytorch学习
Pytorch是搭建神经网络的超强框架，也是近年来最流行的框架。
